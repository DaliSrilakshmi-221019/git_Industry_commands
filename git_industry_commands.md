#----Command Name: git config --global user.name
Syntax:
git config --global user.name "Your Name"

----Purpose---
Sets the username for Git commits on your computer.

-----cmd-2------
Command Name: git config --global user.email

Syntax:

git config --global user.email "your@email.com"

Purpose:
Sets the email used for Git commits.

-----------------cmd-3------------------
Command Name: git config --list

Syntax:
git config --list

Purpose:
Shows all Git configuration settings.

-----------------cmd-4------------------

Command Name: git config --unset

Syntax:
git config --global --unset user.name

Purpose:
Removes a configuration value
